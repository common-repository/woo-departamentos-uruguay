Agrega los 19 departamentos de Uruguay a tu checkout de Woocommerce.

== Description ==
Woocommerce Departamentos Uruguay es un plugin que agrega los 19 departamentos de Uruguay a tu instalación Wordpress con Woocommerce.

Una vez instalado y activado el plugin, se agregarán los 19 departamentos y aparecerán como dropdown en el campo "Región / Provincia" en página de checkout:
Artigas, Canelones, Cerro Largo, Colonia, Durazno, Flores, Florida, Lavalleja, Maldonado, Montevideo, Paysandú, Río Negro, Rivera, Rocha, Salto, San José, Soriano, Tacuarembó y Treinta y Tres.

== Installation ==
Simplemente debes subir/descargar este plugin y activarlo tal como lo haces con cualquier otro plugin de Wordpress. Una vez hayas realizado estos 2 pasos, el plugin ya quedará funcionando.
